# Лабораторная работа №4: Разработка моделей и контроллеров ASP.NET MVC

**Студент:** Сеноженский В.В.  
**Группа:** ИТП--31  
**Вариант:** 16

## Описание

Веб-приложение на ASP.NET Core MVC для управления данными научного портала.

### Реализовано:
-   **Подход Code First:** Модели C# используются для создания структуры базы данных с помощью миграций EF Core.
-   **Инициализация данных:** Middleware для автоматического заполнения базы данных тестовыми данными при первом запуске.
-   **Архитектура MVC:** Созданы контроллеры и представления для CRUD-операций с тремя сущностями.
-   **Отображение связанных данных:** На страницах-списках вместо ID отображаются значимые поля из связанных таблиц (например, названия кафедр).
-   **Кэширование:** Применен атрибут `[ResponseCache]` для кэширования ответов сервера на 272 секунды.
-   **Непрерывная интеграция:** Настроен рабочий процесс GitHub Actions для автоматической сборки проекта.
