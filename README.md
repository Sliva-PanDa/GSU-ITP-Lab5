**Описание предметной области:**

- Кафедры (название, профиль).

**Дополнительные требования к отображению данных:**
- Поиск публикаций по автору, году, типу, журналу, ключевым словам, проекту.

### Анализ технического задания (этап 1)
Технологии: СУБД — MS SQL Server (реляционная модель, скрипты CREATE TABLE/PROC как в "Пример выполнения..."). ЯП — C# .NET 8.0, ASP.NET Core MVC (Razor Views для UI). Доступ к БД — Entity Framework Core 8.0 (миграции, DbContext). Архитектура — Clean Architecture (Domain: entities; Application: DTOs/services/interfaces; Infrastructure: EF/repositories; Web: controllers/views; как в "School Solution" и N-Layer из "Общие архитектуры...").
