# Курсовой проект - ПОРТАЛ НАУЧНЫХ ПУБЛИКАЦИЙ ФАКУЛЬТЕТА

**Группа:** ИТП-31  
**Студент:** Сеноженский Вячеслав Владимирович  
**Вариант:** №16

## Описание проекта
Портал научных публикаций факультета для учета и управления научными работами преподавателей.

## Предметная область
- **Преподаватели** - ФИО, должность, ученая степень, кафедра
- **Кафедры** - название, профиль  
- **Публикации** - название, тип (статья/тезисы/монография), авторы, журнал/конференция, год, DOI/ссылка, файл
- **Журналы/Конференции** - название, рейтинг, издатель, ISSN/ISBN
- **Научные направления** - название, описание
- **Проекты/Гранты** - название, номер, финансирующая организация, сроки, руководитель, участники
- **Связь публикаций с проектами** - публикация, проект

## Функциональные требования
- Поиск публикаций по автору, году, типу, журналу, ключевым словам, проекту
- Формирование списка публикаций преподавателя/кафедры за год
- Отображение индекса Хирша или количества публикаций в журналах Q1-Q2
- Отчет по участию в проектах/грантах

## Технологический стек
- **СУБД:** MS SQL Server
- **Backend:** C# .NET 8.0, ASP.NET Core MVC
- **ORM:** Entity Framework Core 8.0
- **Архитектура:** Clean Architecture

## Структура проекта
- `/db` - SQL скрипты, ER-диаграммы
- `/src` - исходный код приложения
- `/docs` - документация, пояснительная записка
